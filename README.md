# Desplegar la aplicación Local

* Instalar RVM
* Instalar Ruby 2.4.0
* Instalar Rails 5.3.1
* Instalar Postgres

Ejecutar los comando

* bundle install
* rake db:drop db:create db:migrate
* rails server
